Usage:
  imgfilter -i <input.bmp> -o <output.bmp> [options]

Required:
  -i, --input <file>       32‑bit Windows BMP to read
  -o, --output <file>      BMP to write (always 32‑bit)

Filter options (can repeat in any order; each one is a filter stage):
  --contrast <factor>      Adjust contrast by scaling pixel deviation from mid-gray (1.0 = no change)
  --brightness <value>     Add brightness offset to all channels equally (positive or negative integer)
  --blur <radius>          Apply Gaussian blur with the given radius (floating-point or integer)
  --median <radius>        Apply median filter with square window of size (2*radius + 1)
  --edge                   Detect edges using a Sobel-like gradient magnitude
  --offset <r> <g> <b>     Add per‑channel RGB offsets (signed integers)
  --gamma <value>          Apply gamma correction with exponent γ (e.g. 2.2)

Common flags:
  -h, --help               Show this message

Examples:
  # just boost contrast and save
  imgfilter -i in.bmp -o out.bmp --contrast 1.5

  # blur then detect edges
  imgfilter -i photo.bmp -o edges.bmp --blur 3 --edge

  # full pipeline: contrast→blur→gamma→median
  imgfilter -i src.bmp -o dst.bmp \
    --contrast 1.2 \
    --blur 2 \
    --gamma 2.2 \
    --median 1

  # increase brightness and add a blue tint
  imgfilter -i in.bmp -o out.bmp --brightness 20 --rgb 0.9 0.9 1.2
